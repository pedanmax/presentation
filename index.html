Mixins

Mixins can be used in two ways:
Without Arguments and With Arguments.

Without arguments

						
							@mixin textStyle{
								font-size: 24px;
								color: green;
								font-weight: 700;
							}

							.text {
								@include textStyle;
							}

With arguments

						
							@mixin textStyle ($fz, $color, $fw) {
								font-size: $fz;
								color: $color;
								font-weight: $fw;
							}

							.text {
								@include textStyle(16px, red, 500);
							}
							.title {
								@include textStyle(24px, blue, 700);
							}
